# jenr8ed-deploy-kit

Professional automation toolkit for JenR8ed AI projects.

## Principles
- File-System-as-Database (FSAD)
- Zero-bloat, zero-trust
- Single-source staging
- Reusable across all projects

## Structure
- `templates/` - Vercel, README, Notion templates
- `scripts/` - Reusable automation scripts
- `notion/` - Command Center integration
- `docs/` - Guides and handoffs

## Quick Start
1. Clone the repo
2. Run scripts from `scripts/`
3. Use templates for new projects

See `notion/` folder for database structure and integration guide.
